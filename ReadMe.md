# 🎹 AlmaDyne Music — Cinematic & Metal Production
Персональная веб-платформа композитора и звукорежиссера. Проект объединяет страсть к прогрессив-пауэр-металлу и оркестровой киномузыке с современными веб-технологиями.

## 🚀 Технологический стек
* **Frontend:** HTML5, CSS3, JavaScript (ES6+).
* **Backend:** PHP (для обработки почтовых уведомлений).
* **Инструментарий:** * **PHPMailer** — для надежной отправки SMTP-сообщений.
    * **Docker** — для контейнеризации и стабильного деплоя.
    * **Node.js/NPM** — управление скриптами запуска.
* **Деплой:** Render (Web Service via Docker).

## 🛠 Особенности реализации
1.  **Архитектура:** Использован модульный подход в JS и вынос чувствительных настроек в защищенные конфигурационные файлы (не индексируются в Git).
2.  **Валидация:** Реализована комплексная проверка форм на стороне клиента и сервера (PHP `filter_var`).
3.  **Безопасность:** Интеграция с SMTP через защищенное SSL-соединение (порт 465).
4.  **UX/UI:** Адаптивный дизайн, интерактивные формы с обратным отсчетом и защитой от двойного сабмита.
